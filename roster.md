# Pokereign 物种名单（工作文档）

> 这是 Slice 3 的物种规划稿，供编辑。改完 / 补完进化链后告诉我，我据此批量填
> `src/game/data/species.ts` + 扩充招式库 + 配 learnset。

## 约定

- **calc id 是主键**：每只宝可梦的英文 id 必须等于 `@smogon/calc` 的 species id
  （表里「id」列）。属性 / 种族值 / 特性都由 calc 自动取，**不用手填**（见
  `pokedex.ts`）。所以加一只只需：id、中文名、sprite slug（一般同 id）、learnset、
  进化、捕捉档。
- **局内等级 1~12**，进化级段参考 doc §8.2：
  - 三段：Lv.4 第一段、Lv.8 最终段
  - 二段：Lv.6
  - 石头 / 道具进化：等级资格 + `requiredItem`（道具系统是 Phase 2，名单里先占位）
- 「进化触发」列：写 `Lv.N` 表示到级自动进化；写道具名表示需要该道具。
- 属性列只是给你参考（calc 自动取，真实以 calc 为准）。
- **learnset 先留空**，等名单锁定后我从扩充后的招式库统一配；你也可以在「招式备注」
  里点名某条线想要的招牌技能。
